# 🏋️‍♂️ Gym Management – Sistema de Gestión de Gimnasio

Proyecto desarrollado en **Spring Boot** con conexión a **MySQL**, que permite la gestión de usuarios y socios de un gimnasio.

---

## ⚙️ Tecnologías utilizadas
- Java 23  
- Spring Boot 3.5.6  
- Spring Data JPA  
- MySQL 8.0  
- Maven  
- IntelliJ IDEA
